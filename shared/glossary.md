# Glossary

Clinical abbreviations used across the SKIDS protocol library. New
abbreviations get added here as they appear in protocols. Editorial
review fails if a protocol uses an abbreviation that isn't in this
glossary or expanded on first use.

## Adding new abbreviations

When a protocol introduces a new abbreviation, the author must:
1. Expand the abbreviation in full on first use within the protocol
2. Add the abbreviation to this glossary in the appropriate section
3. Commit the glossary update with the protocol

Editorial review checks both: that abbreviations are expanded on first
use within each protocol, and that all abbreviations are in this
glossary.
